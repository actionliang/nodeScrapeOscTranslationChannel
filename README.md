##个人兴趣爱好 用nodejs 转换的一个开源中国 翻译频道 移动浏览版本##

注意: 运行环境需要 win 7 x64 node >= 0.10.x,因为jsdom的依赖库实在前面那个环境编译的.

如果你是linux 用户,你需要把node_modules 删掉,在项目目录下运行

`npm install`
